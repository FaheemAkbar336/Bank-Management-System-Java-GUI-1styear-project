# 💳 Banking Application (Java OOP)

## 📌 Overview

This project is a **Banking Application developed using Java and Object-Oriented Programming (OOP)**. It provides a user-friendly graphical interface to perform basic banking operations such as checking balance, transferring money, paying bills, and managing finances.

---

## 🎯 Features

* 💰 Check Account Balance
* 🔄 Transfer Money
* 🧾 Pay Bills
* 📱 Mobile Load
* 🔐 Secure User Authentication
* 🎨 Interactive GUI using Java Swing

---

## 🛠️ Technologies Used

* **Java** – Core programming language
* **Java Swing** – GUI development
* **OOP Concepts** – Encapsulation, Inheritance, Polymorphism
* **Database (Optional)** – For storing user data
* **MVC Architecture** – Structured and maintainable design

---

## 🧠 Project Objectives

* Provide a simple and easy-to-use banking system
* Ensure secure handling of user data
* Implement core banking functionalities
* Design a scalable system for future enhancements

---

## 🏗️ System Architecture

* **Model-View-Controller (MVC)**
* **Layered Architecture**
* **Client-Server Model (Conceptual)**

---

## 🚀 How to Run

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/banking-application.git
   ```

2. Open in any Java IDE (IntelliJ, Eclipse, NetBeans)

3. Compile and run the main file:

   ```bash
   javac Main.java
   java Main
   ```
